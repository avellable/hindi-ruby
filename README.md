# hindi-ruby
Inspired by: https://github.com/sferik/active_emoji

Write ruby code in hindi.

```
  $ gem install hindi_ruby
```

Create script

```ruby
  require 'hindi_ruby'

  यादी = ['पेन', 'वही']
  यादी.हरएक {|चीज| छापो चीज}
```

```
  $ ruby examples/print_array.rb 
  पेन
  वही
```
